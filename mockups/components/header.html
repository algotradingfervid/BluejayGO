Admin Header Component

This is a demonstration of the Brutalist admin header bar component with breadcrumbs, search, notifications, and user menu.

Features

  • check_circle Full width header with 64px height and 2px black borders
  • check_circle Breadcrumb navigation: Home > Section > Current Page
  • check_circle Search dropdown with recent searches and quick actions
  • check_circle Notifications dropdown with unread badge and mark all read
  • check_circle User menu with avatar, name, email, and navigation options
  • check_circle Brutalist offset shadows (4px 4px 0 0 #000)
  • check_circle All sharp corners - no rounded edges

Search Dropdown

400px width with recent searches and quick actions for creating content.

Notifications

360px width, max 400px height with scrollable list and unread indicators.

User Menu

240px width with profile info, settings, help, and sign out options.

TIP: Click on the search, bell, or user avatar icons in the header to see the dropdown menus.
